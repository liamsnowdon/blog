Post Creator

Use this tool to create blog posts. After filling out all fields, hit the "Copy to JSON" button to copy the post data to the clipboard and add it to the posts.json file for nunjucks. Alternatively, edit a blog post by providing the JSON from data.json or from selecting from the "saved for later" dropdown.
